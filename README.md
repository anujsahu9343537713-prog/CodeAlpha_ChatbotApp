# 🤖 Java Chatbot  
A simple and interactive Chatbot built using Java. This project is created as part of the CodeAlpha Internship to demonstrate basic text-based interaction, keyword detection, and user-response handling.

---

## 🚀 Features
- Understands basic greetings and common phrases  
- Pattern-based keyword response system  
- Runs directly in console (no external libraries)  
- Beginner-friendly clean code structure  

---

## 🛠️ Tech Stack
- **Language:** Java  
- **IDE:** VS Code / IntelliJ / Eclipse  
- **Version Control:** Git & GitHub  

---

## 📌 How to Run the Project
1. Clone or download this repository  
2. Open the project in your Java IDE  
3. Compile the Java file  
4. Run the program using the terminal or IDE run button
